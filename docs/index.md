# Справочник разработчика

## О справочнике

Данный справочник предназначен для **быстрого освежения знаний** по ключевым темам разработки. Материал изложен с практической точки зрения, с акцентом на реализацию в конкретных языках программирования (например, **Python** или **Go**), так как подходы могут существенно отличаться.

>**Важно:** Справочник не является академическим трудом. Он создан как практическое руководство для ежедневной работы и быстрого поиска информации.

## Структуры данных

Раздел охватывает основные структуры данных с примерами реализации и анализа сложности операций.

| Структура данных | Описание |
| ----------------- | ---------- |
| [Массив](./data_structures/array.md) | Базовая структура для хранения упорядоченной коллекции элементов |
| [Множество](./data_structures/set.md) | Коллекция уникальных элементов без повторений |
| [Куча](./data_structures/heap.md) | Деревоподобная структура для эффективного доступа к экстремальным элементам |
| [Хэш-таблица](./data_structures/hash_table.md) | Структура для быстрого поиска по ключу |
| [Связанный список](./data_structures/linked_list.md) | Линейная структура с динамическим выделением памяти |
| [Граф](./data_structures/graph.md) | Структура для моделирования отношений между объектами |
| [Бинарное дерево](./data_structures/binary_tree.md) | Иерархическая структура с двумя потомками на узел |
| [Префиксное дерево](./data_structures/prefix_tree.md) | Дерево для эффективного хранения и поиска строк |
| [LRU-кэш](./data_structures/lru_cache.md) | Кэш с политикой вытеснения наименее недавно используемых элементов |
